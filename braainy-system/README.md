# BraainySystem

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.1.4.

## Running the application

1) Clone the git repository.
2) Access the project folder and run the command 'npm install' to install all dependencies.
3) Once installation is complete run the commant 'npm start'- it will build the application and automatically open the browser on http://localhost:4200/ and the landing page of the application.
